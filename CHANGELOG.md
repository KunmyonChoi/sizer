# Changelog

이 프로젝트의 주요 변경 사항을 기록합니다. 형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를
따르며 [Semantic Versioning](https://semver.org/lang/ko/)을 사용합니다.

## [1.6.0] - 2026-07-23

### Added
- **앱 아이콘** — Finder·응용 프로그램·Spotlight에서 기본(빈) 아이콘 대신 Sizer 아이콘이 표시됩니다.
  로고 SVG로 생성한 `.icns`를 번들에 포함하며, 생성 스크립트(`scripts/make_icon.swift`)는 AppKit만 사용해
  외부 도구 없이 동작합니다(Homebrew 소스 빌드에 추가 의존성 없음).
- **버전 표시** — 메뉴바 팝오버 헤더(`v1.6.0`)와 설정 → 일반 → 정보(`1.6.0 (8)`)에서 확인할 수 있습니다.

## [1.5.1] - 2026-07-23

### Fixed
- **중복 실행 시 결과 파일이 손상되던 문제** — 인스턴스가 둘 실행되면 같은 파일을 동시에 변환하면서
  **같은 임시 파일**(`출력이름.part`)에 두 ffmpeg가 함께 기록해, 결과 mp4가 스트림 없는 재생 불가 파일이 됐습니다.
  임시 파일 경로를 실행마다 고유하게(숨김·UUID) 만들어 근본적으로 차단했습니다. 인스턴스가 몇 개든
  각자 유효한 파일을 만들고 최종 이동만 경합하므로 결과물은 항상 정상입니다. (영상·이미지 모두)
- **단일 인스턴스 가드 추가** — 이미 Sizer가 실행 중이면 폴더 감시를 시작하기 전에 안내 후 종료합니다.
  (`/Applications` 사본과 Homebrew 사본을 동시에 실행하는 경우 등)

## [1.5.0] - 2026-07-22

### Added
- **적응형 임계값(옵션)** — 노이즈 있는 영상에서 정지 구간을 놓치지 않도록 감지 임계값을 검증된 안전 범위
  (−50dB)까지만 자동 완화. 깨끗한 화면 녹화엔 영향 없음. (설정 → 트리밍 → 감지)

### Changed
- 정지/저모션 **감지를 저해상도 프록시(다운스케일+약한 블러)** 에서 수행 — 노이즈 강건성↑, 깨끗한 콘텐츠 무회귀.
- **장면 전환 가드 활성화** — 실제 컷을 가로질러 유지구간을 병합하지 않아 잘라내기 정확도↑.

## [1.4.0] - 2026-07-20

### Added
- **패널 좌/우 위치 선택** — 통합 드롭·셸프 패널을 화면 왼쪽 또는 오른쪽 가장자리에 도킹. 핸들·둥근 모서리·
  펼침 방향·호버 감지 밴드가 위치에 맞게 미러링됩니다(설정 → 드롭 & 셸프).
- **전역 단축키** — 어디서든 지정한 단축키로 패널을 열고 닫습니다. 설정에 단축키 레코더 제공.
  Carbon `RegisterEventHotKey` 기반이라 접근성 권한이 필요 없습니다.

### Changed
- 기본값: 패널 **오른쪽 도킹**, 트리밍 **빨리감기(Beta) + 보수적 민감도**(−58dB / 최소 정지 3.0s / 병합 0.7s).

### Fixed
- 오른쪽 도킹 시 접힌 탭이 핸들이 아닌 트레이 왼쪽을 잘라 보여주던 문제 수정.

## [1.3.0] - 2026-07-20

### Added
- **드롭 타겟 × 파일 셸프 통합 패널** — 화면 왼쪽 가장자리 단일 패널에서 상단은 **변환 드롭존**, 하단은
  **보관 트레이**. 드래그하는 위치(존)에 따라 변환/보관으로 분기하고, 커서가 놓인 존만 강조됩니다.
- **변환 불가 형식 거부** — 변환존에 변환할 수 없는 파일을 끌면 드래그 중 거부 상태(놓기 불가 커서 + 주황 표시)로
  표시되고 드롭이 무시됩니다. 보관 트레이는 모든 파일을 수용.
- **변환 결과 셸프에 자동 추가**(NEW 배지) — 변환이 끝난 결과 파일이 보관 트레이 맨 앞에 얹혀, 바로 원하는
  곳으로 드래그해 옮길 수 있습니다.

### Changed
- 통합 모드가 기본. 설정 **드롭 & 셸프**에서 끄면 예전처럼 자유 부유 **드롭 타겟**과 **파일 셸프**가 분리됩니다.
- 보관 카드를 가로형 격자로 정리하고 통합 패널 레이아웃을 다듬음.

## [1.2.0] - 2026-07-19

### Added
- **파일 셸프(임시 보관대)** — 화면 왼쪽 가장자리에 얇은 탭으로 접혀 있다가 드래그·호버 시 펼쳐지는 임시 파일
  보관대. Finder에서 담아 두고 개별/다중 선택 드래그-아웃으로 원하는 위치로 이동/복사(원본 참조, 이동 시 자동
  제거). 마우스가 있는 화면을 따라다녀 다중 모니터에서도 항상 닿습니다.
- **자동 빨리감기(Beta)** — 정지/저모션 구간을 잘라내지 않고 배속(2/4/8×) 재생해 지루함을 줄이되 맥락은 유지.
  구간별 `setpts`/`atempo`, 배속 구간 오디오 음소거, 우측 상단 »N× 배지.
- **드롭 타겟 변환 후 출력 폴더 자동 열기**(설정, 기본 켬).

### Changed
- 정지/저모션 구간 처리를 **모드 방식**(끔 / 잘라내기 / 빨리감기)으로 재구성하고 **트리밍 기본값을 끔**으로 변경.
- **드롭 타겟 기본 보이기, 파일 셸프 기본 감추기.**
- 플로팅 드롭 타겟·파일 셸프가 **모니터 구성 변경 시 화면 밖으로 사라지지 않도록** 위치를 보정.

## [1.1.0] - 2026-07-19

### Added
- **플로팅 드롭 타겟** — 화면에 떠 있는 콤팩트 드롭 필(HUD 글래스, 항상 위·이동·위치 기억). Finder에서
  파일을 드롭하면 드롭 폴더로 복사되어 자동 변환. 드래그 호버 시 확대·글로우, 드롭 성공 시 카운트 플래시.
- **Homebrew tap** 설치 경로(`brew install KunmyonChoi/tap/sizer`) — 소스 빌드라 개발자 등록/공증 없이
  Gatekeeper 경고 없이 설치.
- **30초 튜토리얼 영상**(hyperframes) + README 프리뷰.

## [1.0.0] - 2026-07-17

### Added
- macOS 메뉴바(트레이) 앱으로 재구성 — Dock 아이콘 없는 agent 앱, 팝오버 메뉴.
- 드롭 폴더 감시(FSEvents + 주기적 재스캔) → 자동 변환.
- 영상 변환: H.264/H.265/VideoToolbox, CRF/Preset/장변 축소/오디오 비트레이트/접미사 설정 노출.
- **움직임 없는 구간 자동 제거** 개선: 민감도 프리셋, 짧은 조각 제거(정확도), 컷 패딩·오디오 페이드(부드러움), 안전장치.
- **이미지 캡처 변환**(ImageIO): AVIF/HEIC/JPEG/PNG, 품질·최대 크기 설정. 스크린샷 기준 70~95% 절감.
- 설정에서 드롭/출력/완료/실패 폴더 변경, 로그인 시 자동 시작(SMAppService), 알림(UserNotifications).
- 최근 변환 목록 클릭 → 결과 재생/열기.
- 변환 중 트레이 아이콘 회전 애니메이션.
- **processed 폴더 자동 정리**(기본 켬, 30일) — 보관 기간 지난 원본 자동 삭제.
- 순수 로직 단위 테스트 + 실제 ffmpeg/ImageIO 통합 테스트.

### 참고
- 이전 버전은 Python `watch_convert.py` launchd 워커였으며 `legacy/`에 보존.
